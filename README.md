# 💼 Adarsh Shetty — Portfolio Website

Welcome to my personal portfolio repository! 🚀  
This website showcases my projects, skills, experience, and contact information in a clean and modern design.

🔗 **Live Website:** https://adarsh-portfolio-iota-ebon.vercel.app  
📂 **Repository:** https://github.com/Adarsh864/Adarsh_portfolio

## ✨ About The Project

This is a responsive and modern portfolio website built using **Next.js**, **TypeScript**, and **Tailwind CSS**.  
It is designed to present my professional profile, technical skills, and featured projects in an elegant and user-friendly way.

### 🔥 Key Features

- 📱 Fully Responsive (Mobile + Desktop)
- ⚡ Fast Performance with Next.js
- 🎨 Clean UI with Tailwind CSS
- 💼 Projects Showcase Section
- 🧠 Skills & Experience Display
- 📬 Contact Section
- ✨ Smooth animations & polished layout

---

## 🧠 Tech Stack

- **Next.js** – React framework
- **TypeScript** – Type-safe JavaScript
- **Tailwind CSS** – Utility-first CSS framework
- **Vercel** – Deployment & Hosting
- **Node.js & npm** – Development environment

## 🚀 Getting Started (Run Locally)

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Adarsh864/Adarsh_portfolio.git
```

### 2️⃣ Navigate into the Folder

```bash
cd Adarsh_portfolio
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Run Development Server

```bash
npm run dev
```

Now open:

```
http://localhost:9002
```

## ✏️ Customization Guide

To update your portfolio:

- Edit personal info → `src/data/`
- Modify sections → `src/components/sections/`
- Update styles → Tailwind classes
- Replace images → `public/` folder
- Change theme/colors → Tailwind config
